# Geetha Balaji's Portfolio

This repository contains my professional portfolio
## 🚀 Features
*   **Professional Portfolio:** Showcases my experience as a Software Engineer and QA Specialist.
*   **Dynamic UI:** Automatically populates skills and project lists using JavaScript.
*   **Interactive Messaging:** A functional message form for visitor inquiries.

## 📁 File Structure
*   `index.html` - Main entry point containing the portfolio sections 
*   `css/index.css` - Custom styling for the layout, experience sections, and image gallery.
*   `js/index.js` - Contains the logic for DOM manipulation 

## 🛠️ How to Run Locally
This project runs entirely in the browser and does not require a local server or environment setup.

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/Geetha82/geetha-balaji-mars
    ```
2.  **Navigate to the Directory:**
    Open the `geetha-balaji-mars` folder on your computer.
3.  **Launch the Application:**
    *   Find the `index.html` file.
    *   **Right-click** the file and select **Open With** (choose Chrome, Firefox, or Edge).
    *   You can also simply **drag and drop** `index.html` into an active browser window.
